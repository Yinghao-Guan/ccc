# Chinese Culture Club Website

A static Next.js landing page for the Chinese Culture Club at Santa Monica College.

## Overview

This project uses Next.js 15 App Router, TypeScript, and React Three Fiber to present the club through a single-page storytelling experience with:

- About
- Events
- Gallery
- Officers
- Joining the club

The design uses custom typography, scroll-based reveal effects, active navigation highlighting, ink-wash visual layers, and a deployed 3D calligraphy brush model.

## Project Structure

- `app/` - App Router entry points, root layout metadata, and global CSS
- `components/` - page sections and interactive layers such as `StoryShell.tsx` and `InkScene.tsx`
- `lib/` - shared choreography and scene helpers
- `public/` - static assets, including `favicon.svg` and the calligraphy brush model
- `out/` - static export output generated by `npm run build`

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Then open `http://localhost:3000`.

Build the static export:

```bash
npm run build
```

## Editing Notes

- Update homepage section copy in `components/`.
- Adjust global visual tokens and shared styling in `app/globals.css`.
- Keep 3D brush behavior in `components/InkScene.tsx` and choreography helpers in `lib/brushChoreography.ts`.
- The favicon is served from `public/favicon.svg` and referenced from the root metadata in `app/layout.tsx`.
- Do not manually edit generated files in `out/`.

## Verification

There is no automated test suite yet. Use the production build as the baseline check:

```bash
npm run build
```

For UI-heavy changes, also verify the homepage in `npm run dev` on desktop and mobile widths.

## Deployment

The site is configured with `output: 'export'` in `next.config.ts`, so `npm run build` generates a static site in `out/`. Deploy that output to static hosting such as GitHub Pages, Netlify, Vercel static hosting, or a traditional web server.
