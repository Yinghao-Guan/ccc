# Chinese Culture Club Website

A static single-page website for the Chinese Culture Club at Santa Monica College.

## Overview

This project is a lightweight frontend built with plain HTML, CSS, and JavaScript. It presents the club through a styled landing page with sections for:

- About
- Events
- Gallery
- Officers
- Joining the club

The design uses custom typography, scroll-based reveal effects, and active navigation highlighting without any framework or build step.

## Project Structure

- `index.html` - page structure and content
- `styles.css` - visual design, layout, typography, and responsive styling
- `main.js` - scroll behavior, reveal-on-scroll animation, and nav state updates

## Local Development

Because this is a static site, you can open `index.html` directly in a browser.

If you prefer running a local server, use any simple static file server from this folder. For example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Editing Notes

- Update copy and section content in `index.html`.
- Adjust colors, spacing, and typography in `styles.css`.
- Modify interaction behavior in `main.js`.

## Deployment

This project can be deployed to any static hosting platform, including GitHub Pages, Netlify, Vercel static hosting, or a traditional web server.
